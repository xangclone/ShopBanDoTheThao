# Shop Bán Đồ Thể Thao - Frontend

## Cài đặt

```bash
npm install
```

## Cấu hình

Tạo file `.env` (tùy chọn):
```
VITE_API_URL=https://localhost:7000/api
```

## Chạy ứng dụng

```bash
npm run dev
```

Ứng dụng sẽ chạy tại: `http://localhost:61620`

## Build production

```bash
npm run build
```

## Cấu trúc

- `src/components/` - React components
- `src/pages/` - Page components
- `src/services/` - API services
- `src/App.jsx` - Main app với routing

## Dependencies chính

- `react-router-dom` - Routing
- `axios` - HTTP client
- `react-toastify` - Notifications
- `tailwindcss` - Styling
